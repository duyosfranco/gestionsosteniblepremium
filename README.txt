Gestión Sostenible v1 – Piloto

Archivos:
- index.html → Escritorio principal con navegación a los módulos
- clientes-firestore.html → Gestión de clientes en tiempo real
- retiros.html → Agenda y rutas con Google Maps
- finanzas.html → Panel financiero y asistente fiscal
- usuarios.html → Administración de cuentas
- configuracion.html → Configuración del perfil corporativo
- temas.html → Personalización de marca
- landing.html → Landing comercial con acceso a la plataforma
- landing-demo.html → Landing demo en modo sólo lectura

Requisitos:
- Proyecto Firebase con Firestore, Authentication (correo y contraseña) y Storage habilitado.
- Reglas de seguridad configuradas para usuarios autenticados.

Deployment:
1. Subir esta carpeta a Firebase Hosting o Netlify.
2. Verificar que index.html se abra correctamente.
3. Login con usuario registrado en Firebase Authentication.
